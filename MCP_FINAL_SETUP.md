   - Select "Kiro" or "MCP"
   - Look for connection errors

   - Type: "MCP: Reconnect Servers"
   - Press Enter

3. **Check Authentication**
   - You may need to authenticate
   - Follow prompts in VS Code
   - Use your Supabase credentials

4. **Verify Config**
   - Open `.kiro/settings/mcp.json`
   - Check URL has your project_ref
   - Check no typos
